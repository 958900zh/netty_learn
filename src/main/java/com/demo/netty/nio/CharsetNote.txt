1. ASCII (American Standard Code of Information Interchange)
    使用 7 bit 来表示一个字符，共计可以表示128个字符

2. ISO-8859-1
    使用 8 bit 表示一个字符，共计可以表示256个字符

3. GB2312
    使用两个字节 16 bit 表示一个汉字

4. GBK
    收录了 GB2312 不包含的部分生僻字

5. GB18030
    包含最全的简体中文

6. BIG5
    繁体中文

7. UNICODE
    采用两个字节来表示一个字符

8. UTF (Unicode Translation Format)
    Unicode是一种编码方式，UTF是一种存储方式，UTF-8是Unicode的一种实现方式

    UTF-16LE(little endian) / UTF-16BE(big endian)

    -> UTF-8
        变长字节表示形式，一般来说，UTF-8使用三个字节表示一个中文
